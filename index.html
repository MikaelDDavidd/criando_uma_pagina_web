Estrutura Básica do HTML

A estrutura básica do HTML é composta por três tags principais que são: HTML, HEAD e BODY. A tag HTML é a principal que envolve as duas próximas tags, a tag HEAD que contém tudo que precisa ser carregado antes da página, como links de estilo, título, o próprio style que é responsável pela estilização da página e o BODY é o corpo da página, a parte que é mostrada para o usuário, como textos, imagens, links e por aí vai.

Falando sobre Tags

As tags representam todos os elementos que podem ser criados no HTML, desde pequenos textos a imagens, links e uma grande variedade de outros elementos. Alguns exemplos de tags são as de texto que vão desde o h1 até o h6, que representam várias variações de títulos de textos, a tag p que representa um parágrafo. As tags são compostas pelo conteúdo do seu elemento mas também por atributos, que vamos ver a seguir.

Atributos das Tags

Os atributos são uma parte essencial no HTML, pois através deles conseguimos referenciar tags criadas, para estilizar, aplicar funções em JavaScript e entre outros. Os atributos estão contidos dentro das tags. Alguns dos atributos principais são: o id, que serve para referenciar tags para o JavaScript, podendo assim agregar funções a um botão, por exemplo, ou salvar um texto de um input. Outro atributo principal é o class, que serve para referenciar as tags para as folhas de estilo dentro do head ou em um arquivo separado, assim possibilitando a estilização dos nossos elementos (tags). Outro atributo muito importante é o style, que nos permite estilizar uma tag dentro da própria tag, o que pode ser muito útil quando precisa experimentar a implementação de um estilo.

Temos também atributos exclusivos de tags como na tag input, nós temos o type que pode definir o tipo de entrada da nossa caixa input, que pode ser um número ou texto entre outros.

Textos

No HTML, os textos são uma parte fundamental na construção de uma página, já que é assim que interagimos com o cliente (usuário). O próprio HTML nos dá uma grande gama de opções de textos para utilizarmos.

Podemos listar alguns deles como os headers ou títulos que são os h1, h2, h3, h4, h5 e h6. Esses são os títulos que temos disponíveis que podem ser citados como uma hierarquia onde varia, por exemplo, o tamanho, desde o h1 sendo o maior e o h6 o menor deles.

Uma das principais tags de texto que utilizamos é a tag p, que é o nosso parágrafo, utilizada para escrever grandes textos como este aqui. Temos também o strong, que deixa o nosso texto em negrito, o i, que deixa o nosso texto em itálico, o span, que destaca uma parte do nosso texto, por exemplo, entre outros.

Listas Ordenadas e Não Ordenads

As listas ordenadas e não ordenadas são elementos fundamentais em HTML para estruturar e organizar o conteúdo de uma página da web. Listas ordenadas são usadas quando a ordem dos itens é significativa, e são representadas pela tag ol, de "Ordered List". Cada item é marcado com a tag li, de "List Item", e por padrão, os itens são numerados sequencialmente. Por outro lado, as listas não ordenadas são usadas quando a ordem dos itens não é importante. Elas são representadas pela tag ul, de "Unordered List". Assim como nas listas ordenadas, cada item é marcado com li, mas os itens não são numerados.

Ambos os tipos de lista podem conter elementos como texto, imagens, links e outros elementos HTML. Eles são úteis para criar menus de navegação, listas de itens de produtos, passos de instruções e muito mais.

Ao usar esses elementos, é importante considerar a semântica e a acessibilidade. Strong elementos são usados para destacar o texto importante, tornando-o visualmente mais forte. Isso é útil para destacar informações cruciais dentro de uma lista, garantindo que sejam facilmente identificáveis pelos usuários, especialmente aqueles que usam leitores de tela ou têm deficiências visuais.

Links

No HTML, os links são elementos fundamentais para conectar diferentes páginas da web e criar uma experiência de navegação interativa para os usuários. Um link é representado pela tag <a>, de "anchor" (âncora), e é usado para direcionar o usuário para outra página da web, recurso, ou até mesmo dentro da mesma página.

Para criar um link, usamos a tag <a> e especificamos o destino do link no atributo href (Hypertext Reference). Por exemplo, para criar um link para a página inicial de um site, podemos escrever <a href="index.html">Página Inicial</a>.

Além do atributo href, podemos adicionar outros atributos aos links para controlar o comportamento, como o atributo target, que define onde a página vinculada será aberta (por exemplo, em uma nova aba ou na mesma aba).

Links podem conter não apenas texto, mas também imagens, tornando-os uma forma versátil de navegação. Eles são essenciais para criar menus de navegação, links para páginas relacionadas, referências externas e muito mais, tornando a web uma rede interconectada de informações e recursos.
